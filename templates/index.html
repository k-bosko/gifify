Gifify your videos

This is our final project for CS6620 Foundations of Cloud Computing at Northeastern University. A Gifify app is a web app that runs on AWS where everybody can upload a video and download a gif file. You will need to sign up first in order to use this app.

How to use:
  1. Upload a video file.
  2. Wait for a gif to generate.

Allowed extensions: mov, mp4, wmv, avi, flv, m4p, mpg, mpeg, mpv, 3gp, 3g2, webm

Allowed file size: Up to 200 MB

The maximum wait time for processing your video file is 15 min.

Please save all your generated gifs locally before logging out. The user data generated during session is deleted after you logout.
